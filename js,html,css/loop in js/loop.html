🔥WHAT IS LOOP IN JAVASCRIPT? 🔥

A loop is used when you want to repeat something again and again.

👉 Instead of writing:
console.log("Hello");
console.log("Hello");
console.log("Hello");

👉 You use a loop:

for(let i = 0; i < 3; i++) {
  console.log("Hello");
}

✔ Same work
✔ Less code
✔ More powerful

🧠 Why loops are important?

Because in real life programming:

• You deal with arrays (lists)
• You process data again and again
• You repeat tasks like:

  • printing numbers
  • checking conditions
  • updating values

🔥 Types of Loops in JavaScript

There are mainly 5 loops:

1. for loop
2. while loop
3. do...while loop
4. for...of loop
5. for...in loop

Plus:
• break
• continue
• Infinite loops

1️⃣ FOR LOOP (Most Important)

Syntax:
for(initialization; condition; update) { }

Example:
for(let i = 0; i < 5; i++) {
  console.log(i);
}

Output:
0 1 2 3 4

Use:
👉 When you know number of iterations

2️⃣ WHILE LOOP

while(condition) { }

Example:
let i = 0;
while(i < 5) {
  console.log(i);
  i++;
}

Use:
👉 When count is unknown

3️⃣ DO...WHILE LOOP

do { } while(condition);

👉 Runs at least once

4️⃣ FOR...OF LOOP

👉 Used for arrays & strings

Example:
let arr = [10, 20, 30];
for(let value of arr) {
  console.log(value);
}

5️⃣ FOR...IN LOOP

👉 Used for objects

Example:
let obj = { name: "Tushar", age: 21 };
for(let key in obj) {
  console.log(key, obj[key]);
}

⚠️ Important Concepts

break: stops loop
continue: skips iteration
Infinite loop: never stops (danger)

🧠 Loop Flow
Start → Condition → Run → Update → Repeat

🧪 Comparison

for → fixed times
while → unknown times
do-while → at least once
for...of → arrays/strings
for...in → objects

🔥 Real Website Example

let items = ["Apple", "Banana", "Mango"];
let ul = document.getElementById("list");

for(let item of items) {
  let li = document.createElement("li");
  li.innerText = item;
  ul.appendChild(li);
}
